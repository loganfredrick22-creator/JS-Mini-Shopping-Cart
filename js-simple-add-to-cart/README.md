# Mini Shopping Cart

A simple interactive web application that allows users to add products to a shopping cart, calculate totals, and display cart items dynamically.

## Features

- Add products with name, price, and quantity.
- Stores products as objects inside an array (`cart`).
- Calculates total price dynamically.
- Formats prices to 2 decimal places for readability.
- Displays all cart items and total in a styled container.
- Input validation to prevent empty or invalid entries.

## Technologies Used

- HTML5 for structure
- CSS3 for styling:
  - Flexbox / Box styling
  - Buttons and inputs
- JavaScript for functionality:
  - Objects and arrays of objects
  - Functions with parameters and return values
  - DOM manipulation (`getElementById`, `.innerHTML`)
  - Loops (`for...of`)
  - Template literals
  - Number formatting with `.toFixed()`

## How to Use

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Enter product name, price, and quantity.
4. Click **"Add to Cart"**.
5. See the cart dynamically updated below the form with total price.
